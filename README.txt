To run the code, cd into the file, run server.py on a CMD first and specify the number of players. 
Then, on another CMD, run client.py, do this step as many times as you specified the number of players. 
After that, enjoy our game!

NOTE!: If an error saying PIL was not found is displayed,
 (    from PIL import Image, ImageTk
ModuleNotFoundError: No module named 'PIL')

Install pillow by running pip install pillow.
